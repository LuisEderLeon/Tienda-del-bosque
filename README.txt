# Tienda del bosque

Este proyecto es una demostración de cómo se vería una tienda virtual del mundo del Bosque Mágico. Para más información sobre El Bosque Mágico, visita: https://www.wattpad.com/story/369510349?utm_source=android&utm_medium=link&utm_content=share_writing&wp_page=create&wp_uname=Fotatata

En su estado actual, esta tienda virtual tiene funcionalidad limitada pero existente. Se planea expandirla en el futuro, finalizando algunas páginas aún no implementadas, como los enlaces en el perfil.

Para ver su funcionamiento actual existen 2 opciones: la mas sencilla es ir al siguiente enlace: https://luisederleon.github.io/Tienda-del-bosque/
La otra forma es la siguiente: descarga el proyecto, extrae el archivo ZIP y abre "index.html". Desde allí, puedes acceder a las otras páginas.

Este proyecto fue realizado con HTML, CSS y Javascript, sin el uso de librerías externas ni Bootstrap.


## Menu principal

En el menu principal se puede evidenciar un listado con todos los items a la venta, cada uno con su precio y una opción para marcarlo como Favorito, con su debida animación. Al hacer click en alguno de los productos se redirige a la pagina de dicho producto. Los favoritos se guardan en la sesion temporal.

## Descriptores

Al ingresar al descriptor de un objeto, se podra evidenciar una imagen del mismo en tamaño mas grande, asi como una descripción de como funciona el objeto, basado en conceptos de pseudo-ciencia.
El boton "Comprar Ya" añade el objeto al carrito en caso de no existir aun, o añade una unidad extra en caso de ya existir. Los favoritos son funcionales y se actualizan de la forma debida.

## Carrito

En la sección del carrito se puede ver una lista con diferentes objetos que el usuario ha elegido. Existen 3 botones, uno para añadir mas objetos al carrito, uno para quitar un objeto del carrito, y otro para ir al descriptor del objeto, los dos primeros con animaciones.

## Perfil

La página del perfil no contiene funcionalidad real al igual que las otras, pero si tiene diferentes placeholders para diversas funciones pensadas a futuro.
