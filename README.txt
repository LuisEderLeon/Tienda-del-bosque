# Tienda del bosque

Este proyecto es una demostración de cómo se vería una tienda virtual del mundo del Bosque Mágico. Para más información sobre El Bosque Mágico, visita: https://www.wattpad.com/story/369510349?utm_source=android&utm_medium=link&utm_content=share_writing&wp_page=create&wp_uname=Fotatata

En su estado actual, esta tienda virtual solo muestra el concepto y carece de funcionalidad real. Se planea expandirla en el futuro, implementando JavaScript y finalizando algunas páginas aún no implementadas, como los enlaces en el perfil y el botón "Comprar ya!" de los ítems individuales.

Para ver su funcionamiento actual, descarga el proyecto, extrae el archivo ZIP y abre "inicio.html". Desde allí, puedes acceder a las otras páginas.

Este proyecto fue realizado con HTML y CSS puros, sin el uso de librerías externas ni Bootstrap.


## Menu principal

En el menu principal se puede evidenciar un listado con todos los items a la venta, cada uno con su precio y una opción para marcarlo como Favorito, con su debida animación. Al hacer click en alguno de los productos se redirige a la pagina de dicho producto

## Descriptores

Al ingresar al descriptor de un objeto, se podra evidenciar una imagen del mismo en tamaño mas grande, asi como una descripción de como funciona el objeto, basado en conceptos de pseudo-ciencia.

## Carrito

En la sección del carrito se puede ver una lista con diferentes objetos, aunque sin funcionalidad real. Existen 3 botones, uno para añadir mas objetos al carrito, uno para quitar un objeto del carrito, y otro para ir al descriptor del objeto, los dos primeros con animaciones.

## Perfil

La página del perfil no contiene funcionalidad real al igual que las otras, pero si tiene diferentes placeholders para diversas funciones pensadas a futuro.
